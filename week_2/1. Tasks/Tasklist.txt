1. Определитель високосного года
	В Григорианском календаре високосный год определяется по следующему алгоритму: 
	високосным является каждый четвертый год, но каждый сотый високосным не является, 
	при этом каждый 400-й год все таки високосный. Т.е. 1823 - не високосный, 1824 - високосный, 
	1900 - не високосный, 2000 - високосный.
	Напишите программу, предлагающую пользователю ввести год, и выводящую информацию о его "високосности"
	Предусмотрите проверку корректности ввода и вывод сообщений об ошибках.

2. Комбинатор
	Напишите приложение, подсчитывающее количество размещений,
	перестановок и сочетаний для последовательностей, параметры которых заданы пользователем.

3. Файловый сплиттер
	Напишите приложение, считывающее строки из файла и сохраняющее каждую строку в новый файл. Имена файлов должны состоять из первого слова в строке и расширения .txt

4. Файловый фильтр
	Напишите приложение, фильтрующее файл по первому слову в строке
	и выводящее результат на консоль.

5. Парсер ini формата
	Напишите приложение, предназначенное для считывания файлов в ini формате.
	Файл состоит из секций, имя секции задаётся в двойных квадратных скобках.
	Каждая секция содержит переменные в формате <имя>=<значение>

	Пример:
		[[config]]
		controlPort=8900
		idleTimeout=20
		[[admin]]
		login=petrovich
		email=petrovich@zavod.ru
		password=secret

	После анализа файла его содержимое должно быть представлено в виде
	словаря, ключами которого являются имена секций, а значениями - словари переменных.
	Выведите на консоль количество секций и кол-во переменных в каждой секции.

6. Калькулятор последовательностей
	Напишите приложение, считывающее из файла последовательность чисел
	и выводящее их сумму, произведение, среднее арифметическое и факториал каждого числа. 

7. Обнаружитель числовых взаимосвязей
	Напишите приложение, считывающее из файла последовательность чисел и находящую
	связи между ними.
	Виды искомых связей:
	сумма,
	разность,
	произведение,
	частное,
	степень,
	логарифм по основанию 2.
	Выведите найденные комбинации чисел и вид зависимости.
	Пример:
	Исходная последовательность:
	25,2357,32,5,75,2356,3,1
	Вывод:
	2356,1,sum,2357
	25,3,mul,75
	32,log2,5
	и т.д.

8. Умножитель матриц
	Напишите приложение, выполняющее умножение двух матриц, заданных в отдельных файлах.
	Результат выведите на консоль. При некорректных данных обеспечьте диагностику.

9. Морской бой
	Напишите приложение, для игры в морской бой с компьютером.
	Игровое поле представляет собой матрицу 10х10 с индексами от 1 до 10 по горизонтали
	и от a до k по вертикали.
	У каждого противника имеются следующие корабли:
		1 четырёхтрубный (4 смежные клетки)
		2 трёхтрубных
		3 двухтрубных
		4 однотрубных

	В начале игры пользователь вводит координаты своих кораблей.
	Пример:
		"Введите координаты четырёхтрубника":b3-b6

	Затем случайным образом формируется карта компьютера.
	Подумайте, в каком формате удобнее хранить карту.

	Игроки поочерёдно делают ходы, компьютер ходит случайным образом.
	Выигрывает тот, кто первым потопит корабли противника.

	В качесиве дополнительного задания попробуйте реализовать оптимальную
	стратегию для компьютера.
